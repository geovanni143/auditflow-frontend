# AuditFlow Frontend

AuditFlow is a fullstack MVP for managing security audit projects and vulnerability findings.

This repository contains the **Frontend** built with **Next.js**, **React**, **TypeScript**, **Tailwind CSS** and the **App Router**.

The application connects to a Java/Spring Boot backend deployed separately.
